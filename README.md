# Conference Ticket Generator

## Overview

The Conference Ticket Generator is a single-page web application that allows users to generate a personalized ticket for the "Coding Conf 2025". Users can upload their avatar, enter their name, email, and GitHub username, and instantly receive a visually styled ticket with their details and a unique ticket number.

## Technologies Used

- **HTML5**: Structure of the web page and form.
- **CSS3**: Custom styling, responsive layout, flexbox, and transitions for smooth animations.
- **JavaScript (ES6)**: Handles form interactions, image upload preview, dynamic ticket generation, and page transitions.
- **SVG & PNG Assets**: Custom icons, backgrounds, and ticket graphics.

## Features

- Avatar upload with live preview and change/remove options.
- Form fields for name, email, and GitHub username.
- Animated transition from the form to the ticket confirmation.
- Ticket displays event details, user info, avatar, logo, and a randomly generated ticket number (rotated for style).
- Responsive design and visually appealing layout.

## End Goal

The goal of this project is to provide a fun, interactive way for users to generate and preview their conference ticket, simulating a real-world event registration experience. It demonstrates modern front-end development practices, including dynamic DOM manipulation, CSS transitions, and user-friendly form handling.
